# 🤖 Ravichandra Reddy — Portfolio Website

> **Live Site:** [ravichandra260.github.io](https://ravichandra260.github.io)

A futuristic Iron Man / Jarvis-themed personal portfolio built with pure HTML, CSS, and JavaScript. No frameworks. No dependencies. Just pure frontend craft.

---

## ✨ Features

- 🎬 **Boot-up Loader** — System initialization animation with percentage counter
- 🌐 **Particle Network** — Animated connected dots background
- ✨ **Custom HUD Cursor** — Glowing cursor with trail effect
- ⌨️ **Typing Role Animation** — Cycles through titles automatically
- 📊 **Animated Skill Bars** — Fill on scroll with IntersectionObserver
- 🔢 **Counter Animation** — Stats count up on viewport entry
- 🤖 **J.A.R.V.I.S. Chatbot** — AI bot that answers questions about Ravi
- 📡 **Live Clock HUD** — Real-time clock in corner overlay
- 🔲 **Scroll Reveal** — Cards animate in as you scroll
- 📈 **Scroll Progress Bar** — Blue/gold gradient at top of page
- 🎯 **Fully Clickable** — All links, emails, phone, GitHub, LinkedIn work
- 📱 **Responsive** — Works on mobile and desktop
